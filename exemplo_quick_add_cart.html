📋 Como Implementar

1. Estrutura HTML obrigatória

Cada produto da listagem precisa ter a classe listagem-item e uma classe prod-id-{ID_DO_PRODUTO}: 

<div class="listagem-item prod-id-373398725">
    <!-- conteúdo do produto -->
</div>

2. Configuração (antes do script)

<script>
window.QuickAddCartConfig = {
    active: true,

    // Textos
    addButtonText:     'Adicionar ao carrinho',
    outOfStockText:    'Sem estoque',
    selectHintText:    'Selecione uma opção',
    popupTitle:        'Produto adicionado ao carrinho!',
    popupKeepText:     'Continuar comprando',
    popupCheckoutText: 'Finalizar compra',

    // Cores (substitui apenas as informadas; demais mantêm o padrão)
    colors: {
        primary:      '#1f3a5f',
        primaryHover: '#16305a',
        success:      '#16a34a'
    }
};
</script>

3. Carregar o script

<script src="quick_add_cart.js"></script>

Comportamento das variações

  • Grupos com grid.value_for_display igual a "Cor" → exibem bolinha colorida usando value.codes.primary
  • Demais grupos → exibem quadradinho com o texto do valor
  • Quantidade máxima é limitada ao inventory.available_quantity da variação selecionada
  • O botão fica desabilitado até que todos os grupos de variação tenham uma seleção
  • Adição ao carrinho via GET /carrinho/produto/{skuId}/adicionar/{quantidade}
  • Após sucesso exibe popup com "Continuar comprando" ou "Finalizar compra" → /carrinho/index
