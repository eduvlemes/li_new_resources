Referência de configuração

Todas as opções disponíveis no objeto window.M2CalculatorConfig:

Propriedade Tipo Descrição
onQuantityCalculated Function Callback disparado ao confirmar as medidas. Recebe a quantidade (inteiro) como argumento. Principal ponto de integração com o carrinho.
products Array Lista de objetos de configuração por produto (ver detalhes abaixo).
priceSelector String Seletor CSS do elemento que contém o preço unitário. Padrão: .principal [data-sell-price].
priceAttribute String Atributo HTML do qual o preço é lido. Padrão: data-sell-price.
insertAfterSelector String Seletor CSS do elemento após o qual o widget será inserido. Se vazio, insere no final do <body>.
title String Título exibido no topo do widget.
widthLabel / heightLabel String Rótulos dos campos de largura e altura.
buttonText String Texto do botão de confirmação.
colors Object Objeto de cores para personalizar a aparência. Merge parcial suportado.

Propriedades de cada objeto em products[]:

Propriedade Tipo Descrição
condition String Expressão JS avaliada em runtime. Ex: "window.PRODUTO_ID == 399627934". Omita para ativar em qualquer página.
quantityRatio Number Unidades por m². Ex: 1000 → 1,50 m² = 1500 unidades. Fórmula: Math.ceil(largura × altura × quantityRatio).
quantityFormula Function Fórmula personalizada: function(width, height) { return qty; }. Substitui quantityRatio quando definida.
minWidth / maxWidth Number Limites de largura em metros. Exibe erro de validação se ultrapassados.
minHeight / maxHeight Number Limites de altura em metros. Exibe erro de validação se ultrapassados.
widthStep Number Incremento do campo de largura (atributo step do input). Ex: 0.01, 0.05, 0.10, 1. Padrão: 0.01.
heightStep Number Incremento do campo de altura (atributo step do input). Ex: 0.01, 0.05, 0.10, 1. Padrão: 0.01.
priceSelector / priceAttribute String Sobrescreve os seletores de preço globais apenas para este produto.

Código de implementação

Adicione o bloco abaixo na página do produto, antes de carregar o script. Ajuste condition, quantityRatio e onQuantityCalculated conforme a loja. 

<!-- 1. Configuração — inserir ANTES do script -->
<script>
window.M2CalculatorConfig = {

    // Callback disparado ao confirmar as medidas.
    // Adapte para atualizar o campo de quantidade do seu tema.
    onQuantityCalculated: function(quantity) {
        var qtyInput = document.querySelector('[name="qty"]');
        if (qtyInput) {
            qtyInput.value = quantity;
            qtyInput.dispatchEvent(new Event('change', { bubbles: true }));
        }
    },

    // Seletor do elemento após o qual o widget será inserido
    insertAfterSelector: '#comprar',   // Ex: botão "Comprar"

    // Seletor e atributo do preço unitário
    priceSelector: '.principal [data-sell-price]',
    priceAttribute: 'data-sell-price',

    // Configuração por produto
    products: [
        {
            condition: 'window.PRODUTO_ID == 399627934',
            quantityRatio: 1000,   // 1 m² = 1000 unidades
            minWidth:  0.10,
            maxWidth:  5.00,
            minHeight: 0.10,
            maxHeight: 3.00,
            widthStep:  0.01,      // incremento do input de largura
            heightStep: 0.01,      // incremento do input de altura
        },
        {
            condition: 'window.PRODUTO_ID == 112233445',
            quantityRatio: 500,    // 1 m² = 500 unidades
            minWidth:  0.20,
            maxWidth: 10.00,
            minHeight: 0.20,
            maxHeight:  4.00,
            widthStep:  0.05,
            heightStep: 0.05,
        },
    ],

    // Personalização visual (opcional — merge parcial)
    colors: {
        primary: '#2563eb',
        primaryHover: '#1d4ed8',
    },
};
</script>

<!-- 2. Script do plugin -->
<script src="resources/m2_calculator.js"></script>

Para usar uma fórmula de quantidade customizada (ex: arredondamento diferente, múltiplos de caixa), substitua quantityRatio por quantityFormula: 

// Exemplo: arredondar para múltiplos de 50
quantityFormula: function(width, height) {
    var raw = width * height * 1000;
    return Math.ceil(raw / 50) * 50;
},
