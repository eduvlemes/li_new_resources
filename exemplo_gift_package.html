🎁 Gift Package Plugin

Adicione uma embalagem de presente ao carrinho com campo para mensagem personalizada

Plugin em ação

O plugin abaixo é renderizado no elemento com id="plugin-target" via targetSelector. Clique em Adicionar para simular a adição ao carrinho — o campo de mensagem aparecerá automaticamente.

Como usar

Defina window.GiftPackageConfig antes de carregar o script. Implemente os callbacks onAddClick e onRemoveClick para integrar com a API do seu carrinho. Chame plugin.setInCart(true/false) dentro dos callbacks para confirmar a ação e atualizar a UI. 

<script>
window.GiftPackageConfig = {
    title: "Vai presentear? Dê um upgrade com a Gift Bag",
    description: "Aviso: A sacola é enviada à parte para você montar.",
    productName: "Insider Gift Bag",
    productPrice: "R$ 9,00",
    productImageUrl: "https://exemplo.com/gift-bag.png",
    targetSelector: "#gift-package-slot", // onde o plugin será inserido

    onAddClick: function(plugin) {
        // Chame aqui a API do seu carrinho para adicionar o produto
        adicionarAoCarrinho({ id: "gift-bag", preco: 9.00 })
            .then(function() {
                plugin.setQuantity(plugin.getQuantity() + 1); // confirma: atualiza contador
            })
            .catch(function() {
                alert("Não foi possível adicionar. Tente novamente.");
            });
    },

    onRemoveClick: function(plugin) {
        // Chame aqui a API do seu carrinho para remover uma unidade
        removerDoCarrinho({ id: "gift-bag" })
            .then(function() {
                plugin.setQuantity(plugin.getQuantity() - 1); // confirma: decrementa ou volta ao botão
            });
    },

    onCardMessageChange: function(message, plugin) {
        // Chamada a cada tecla digitada na mensagem do cartão
        console.log("Mensagem do cartão:", message);
    },

    colors: {
        addButtonBackground: "#000000",
        addButtonHover: "#222222"
    }
};
</script>
<script src="resources/gift_package.js"></script>

Referência da API Pública

Após a inicialização, o plugin fica disponível em window.GiftPackage.

// Define a quantidade no carrinho e atualiza a interface
// Quantidade 0 reverte o seletor de volta para o botão "Adicionar"
window.GiftPackage.setQuantity(2)

// Retorna a quantidade atual no carrinho
window.GiftPackage.getQuantity()

// Retorna true se quantidade > 0
window.GiftPackage.isInCart()

// Retorna o texto atual digitado no campo de mensagem do cartão
window.GiftPackage.getCardMessage()

// Define o texto do campo de mensagem do cartão
window.GiftPackage.setCardMessage("Feliz aniversário! 🎂")

// Remove o plugin completamente do DOM
window.GiftPackage.destroy()

// Reinicializa o plugin do zero
window.GiftPackage.reinit()
