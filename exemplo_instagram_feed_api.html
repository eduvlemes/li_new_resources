Instagram Feed Plugin

Exibe o feed diretamente da API Graph do Instagram — sem intermediários, usando seu próprio token de acesso.

dataSource: instagram

Demo ao vivo

Informe um Long-Lived Access Token do Instagram para carregar o feed em tempo real. O token nunca sai do seu navegador — a requisição vai direto para graph.instagram.com.

Atenção: Este campo é apenas para testes locais. Em produção, jamais exponha o token em HTML público sem restringir o domínio no painel do seu App do Facebook.

API Pública

Após carregar o feed, você pode controlá-lo via window.InstagramFeed:

// Recarrega o feed mantendo a config atual
window.InstagramFeed.reload();

// Atualiza config e reinicializa
window.InstagramFeed.updateConfig({
    columns: { desktop: 3, tablet: 2, mobile: 1 }
});

// Reinicializa sem alterar a config
window.InstagramFeed.reinit();

// Remove completamente do DOM
window.InstagramFeed.destroy();

Como obter o Long-Lived Access Token

O Long-Lived Token dura 60 dias. Para uso contínuo em produção, implemente um endpoint server-side que faça o refresh antes do vencimento usando a rota GET /refresh_access_token da API do Instagram.
  1. Acesse developers.facebook.com e crie (ou abra) um App do tipo Consumer ou Business.
  2. Adicione o produto Instagram Basic Display ao seu App.
  3. Em Basic Display → User Token Generator, clique em Generate Token para a sua conta de teste.
  4. Copie o Short-Lived Token gerado (válido por 1 hora) e troque-o por um Long-Lived Token com a chamada abaixo (substitua os valores): 
    # Trocar Short-Lived por Long-Lived Token (válido 60 dias)
GET https://graph.instagram.com/access_token
    ?grant_type=ig_exchange_token
    &client_secret={APP_SECRET}
    &access_token={SHORT_LIVED_TOKEN}
  5. Cole o access_token retornado no campo acima ou em window.InstagramFeedConfig.accessToken no seu HTML.
  6. Renove antes do vencimento (máx 60 dias) com: 
    # Renovar Long-Lived Token (a partir do dia 25 antes de expirar)
GET https://graph.instagram.com/refresh_access_token
    ?grant_type=ig_refresh_token
    &access_token={LONG_LIVED_TOKEN}

Configuração completa — modo Instagram direto

Defina window.InstagramFeedConfig antes de carregar o script:

window.InstagramFeedConfig = {

    // ── FONTE DE DADOS ───────────────────────────────────────
    dataSource:  'instagram',          // 'behold' (padrão) | 'instagram'
    accessToken: 'SEU_LONG_LIVED_TOKEN', // obrigatório quando dataSource === 'instagram'

    // ── INSERÇÃO ────────────────────────────────────────────
    insertSelector: '#meu-container',   // seletor CSS do container
    insertMethod:   'beforeend',         // beforeend | afterbegin | beforebegin | afterend

    // ── HEADER DE PERFIL ────────────────────────────────────
    showHeader: true,
    profile: {
        displayName:    '',            // sobrescreve o username do Instagram
        showStats:      true,
        postsLabel:     'publicações',
        followersLabel: 'seguidores',
        followingLabel: 'seguindo',
    },
    followButtonText: 'Seguir',
    followButtonUrl:  '',              // ex: 'https://instagram.com/seu_usuario'

    // ── GRID ────────────────────────────────────────────────
    columns: { desktop: 4, tablet: 3, mobile: 2 },
    gap:              '3px',
    imageSize:        'medium',        // 'small' | 'medium' | 'large' | 'full'

    // ── ÍCONES E OVERLAY ────────────────────────────────────
    showVideoIcon:    true,
    showCarouselIcon: true,
    showHoverOverlay: true,

    // ── LIGHTBOX ────────────────────────────────────────────
    showLightbox:      true,
    lightboxImageSize: 'large',
    lightboxLinkText:  'Ver no Instagram',

    // ── TEXTOS ──────────────────────────────────────────────
    loadingText: 'Carregando feed...',
    errorText:   'Não foi possível carregar o feed.',
    emptyText:   'Nenhum post encontrado.',

    // ── CORES (todas opcionais) ──────────────────────────────
    colors: {
        sectionBackground: 'transparent',
        headerText:        '#262626',
        headerSubText:     '#737373',
        followButton:      '#0095f6',
        followButtonText:  '#ffffff',
        postOverlay:       'rgba(0,0,0,0.5)',
        lightboxOverlay:   'rgba(0,0,0,0.85)',
        iconColor:         '#ffffff',
        // ... todos os tokens em resources/instagram_box_feed.js → DEFAULT_CONFIG.colors
    },
};

Behold.so vs API Direta — comparativo

Escolha a fonte de dados de acordo com o seu cenário:

┌──────────────────────┬──────────────────────┬──────────────────────┐
│                      │      Behold.so        │   Instagram direto   │
├──────────────────────┼──────────────────────┼──────────────────────┤
│ Token expira?        │ Não                  │ Sim (60 dias)        │
│ URLs de imagem       │ CDN estável (Behold)  │ CDN do Instagram*    │
│ Bio / website        │ Sim                  │ Não (Basic Display)  │
│ Reqs HTTP p/ carga   │ 1                    │ 2 (perfil + mídia)   │
│ Custo                │ Plano gratuito lim.  │ Grátis (API)         │
│ Dependência externa  │ Behold.so            │ Nenhuma              │
│ Necessita servidor?  │ Não                  │ Para refresh do token│
└──────────────────────┴──────────────────────┴──────────────────────┘

* URLs de mídia da API do Instagram expiram após alguns dias.
  Para cachê-las de forma permanente, sirva-as via proxy server-side.
