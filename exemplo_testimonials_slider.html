Testimonials Slider

Plugin de depoimentos responsivo com slider em Vanilla JS — sem dependências externas.

Como usar (uso básico)

Carregue o script na página. O widget é inserido automaticamente no final do <body>. Nenhuma dependência — zero jQuery, zero bibliotecas externas. 

<!-- Basta incluir o script -->
<script src="resources/testimonials_slider.js"></script>

Personalização completa via window.TestimonialsSliderConfig

Defina o objeto de configuração antes do script. Você pode sobrescrever apenas as propriedades que quiser — o restante usa os valores padrão.

Exemplo com depoimentos reais e tema escuro
// Definir ANTES de carregar o script
window.TestimonialsSliderConfig = {
    title: "BEM-ESTAR NÃO É META. É ESTADO.",
    subtitle: "Veja o que nossos clientes e especialistas dizem",
    insertSelector: "#secao-depoimentos",
    insertMethod: "afterbegin",

    autoplay: true,
    autoplayDelay: 5000,
    loop: true,
    showArrows: true,
    showDots: true,

    visibleCards: {
        desktop: 3,
        tablet: 2,
        mobile: 1
    },

    testimonials: [
        {
            quote: "O produto me ajuda a repor o que preciso em um único ritual matinal.",
            author: "Carlos Silva",
            role: "Empresário",
            photo: "https://exemplo.com/foto-carlos.jpg"
        },
        // ... mais depoimentos
    ],

    // Tema escuro
    colors: {
        sectionBackground: "#0d0d0d",
        titleColor: "#ffffff",
        quoteBoxBackground: "#1c1c1c",
        quoteBoxBorder: "#2a2a2a",
        quoteTextColor: "#cccccc",
        photoOverlayGradient: "rgba(0,0,0,0.0), rgba(0,0,0,0.9)",
        authorNameColor: "#ffffff",
        dotActive: "#ffffff"
    }
};

API pública

O plugin fica disponível em window.TestimonialsSlider após inicialização.

// Ir para um slide específico (0-indexed)
window.TestimonialsSlider.goTo(2);

// Navegar manualmente
window.TestimonialsSlider.next();
window.TestimonialsSlider.prev();

// Controlar o autoplay
window.TestimonialsSlider.stopAutoplay();
window.TestimonialsSlider.startAutoplay();

// Remover o widget da página
window.TestimonialsSlider.destroy();

Referência completa de opções

Todas as propriedades disponíveis em window.TestimonialsSliderConfig:

window.TestimonialsSliderConfig = {
    active: true,                  // false desativa o plugin completamente

    // Inserção no DOM
    insertSelector: "body",         // Seletor CSS do elemento pai
    insertMethod: "beforeend",     // beforeend | afterbegin | beforebegin | afterend

    // Título
    showTitle: true,
    title: "O QUE NOSSOS CLIENTES DIZEM",
    subtitle: "",                  // Opcional

    // Depoimentos
    testimonials: [
        {
            quote: "Texto do depoimento...",
            author: "Nome Sobrenome",
            role: "Cargo ou título",
            photo: "https://..."       // URL da foto (opcional)
        }
    ],

    // Slider
    autoplay: true,
    autoplayDelay: 5000,           // ms entre slides automáticos
    loop: true,
    showArrows: true,
    showDots: true,
    visibleCards: {
        desktop: 3,              // >= 1024px
        tablet: 2,               // >= 600px
        mobile: 1                // < 600px
    },
    cardGap: 24,                  // px entre os cards

    // Cores (sobrescreva apenas as que quiser)
    colors: {
        sectionBackground: "#f5f5f5",
        titleColor: "#111111",
        subtitleColor: "#666666",
        quoteBoxBackground: "#ece9e3",
        quoteBoxBorder: "transparent",
        quoteBoxShadow: "rgba(0,0,0,0.04)",
        quoteMarkColor: "#aaaaaa",
        quoteTextColor: "#2a2a2a",
        photoBoxBackground: "#1a1a1a",
        photoBoxBorder: "transparent",
        photoOverlayGradient: "rgba(0,0,0,0.0), rgba(0,0,0,0.85)",
        authorNameColor: "#ffffff",
        authorRoleColor: "rgba(255,255,255,0.72)",
        authorInitialsColor: "#555555",
        arrowBackground: "#ffffff",
        arrowBorder: "#e0e0e0",
        arrowIcon: "#333333",
        arrowHover: "#f0f0f0",
        dotActive: "#111111",
        dotInactive: "#cccccc"
    },

    // Dimensões e espaçamentos
    sectionPadding: "72px 20px",
    cardBorderRadius: "16px",
    photoAspectRatio: "3/4",

    // Classe CSS extra na <section> para customização externa
    containerClass: ""
};
