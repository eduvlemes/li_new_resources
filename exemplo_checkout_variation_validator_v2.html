🛒 Checkout Variation Validator v2 infoBoxes

Demonstração da funcionalidade de popups informativos vinculados a variações detectadas no carrinho.

ℹ️ Como funciona o infoBoxes

Ao clicar em Finalizar Compra, o plugin detecta os modelos no carrinho. Se algum deles contiver qualquer palavra de triggerWords configurada, um botão informativo aparece no rodapé do modal de validação. Ao clicar nesse botão, um popup secundário exibe o conteúdo HTML definido em content.

⚙️ Configuração Utilizada Nesse Exemplo

infoBoxes ativos

infoBox #1 — iPhone / Apple

triggerWords: ['IPHONE', 'APPLE']

actionText: "Ver guia de compatibilidade iPhone"

title: "Compatibilidade — Linha iPhone"

Aparece quando qualquer variação detectada contiver "IPHONE" ou "APPLE".

infoBox #2 — Samsung / Galaxy

triggerWords: ['SAMSUNG', 'GALAXY']

actionText: "Ver guia de compatibilidade Samsung"

title: "Compatibilidade — Linha Samsung"

Aparece quando qualquer variação detectada contiver "SAMSUNG" ou "GALAXY".

Código de Configuração

window.CheckoutVariationValidatorConfig = { active: true, pageSelector: '.pagina-carrinho', productSelector: '.tabela-carrinho [data-produto-id] .produto-info > a', buttonSelector: '.finalizar-compra .acao-editar .botao.principal.grande', variations: [ 'IPHONE AIR', 'IPHONE 17 PRO', 'IPHONE 15', 'SAMSUNG S24' ], // ─── NOVO: infoBoxes ─────────────────────────────────────── infoBoxes: [ { active: true, triggerWords: ['IPHONE', 'APPLE'], // array de palavras-gatilho title: 'Compatibilidade — Linha iPhone', actionText: 'Ver guia de compatibilidade iPhone', content: `<p>...seu HTML informativo...</p>` }, { active: true, triggerWords: ['SAMSUNG', 'GALAXY'], title: 'Compatibilidade — Linha Samsung', actionText: 'Ver guia de compatibilidade Samsung', content: `<p>...seu HTML informativo...</p>` } ], // ────────────────────────────────────────────────────────── messages: { single: 'Você está comprando um item para:{models}<br>Deseja prosseguir?', multiple: 'Você está comprando itens para:{models}<br>Deseja prosseguir?', title: 'Confirmar Modelos', confirmButton: 'Sim, está correto', cancelButton: 'Vou revisar' }, debug: true };
