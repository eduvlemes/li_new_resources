FAQ Accordion

Plugin de perguntas frequentes com accordion animado e layout de duas colunas.

Inclusão básica

<!-- Apenas inclua o script. O plugin se insere automaticamente. --> <script src="resources/faq_accordion.js"></script>

Personalização (antes do script)

<script> window.FaqAccordionConfig = { title: 'Dúvidas?', description: 'Estamos aqui para ajudar!', openFirstItem: true, allowMultipleOpen: false, items: [ { question: 'Pergunta 1', answer: 'Resposta 1' }, { question: 'Pergunta 2', answer: 'Resposta 2' } ], colors: { titleColor: '#0c2928', itemBorderColor: '#208d58', buttonBackground: '#208d58', buttonHoverBackground: '#0c2928' } }; </script> <script src="resources/faq_accordion.js"></script>

API pública

Método Descrição
window.FaqAccordion.openItem(index) Abre o item de índice index (começa em 0).
window.FaqAccordion.closeItem(index) Fecha o item de índice index.
window.FaqAccordion.closeAll() Fecha todos os itens do accordion.
window.FaqAccordion.updateConfig(obj) Atualiza a configuração e reinicializa o plugin.
window.FaqAccordion.reinit() Reinicializa o plugin do zero.
window.FaqAccordion.destroy() Remove completamente o plugin e seus estilos do DOM.

Opções de configuração (DEFAULT_CONFIG)

Propriedade Padrão Descrição
active true Ativa ou desativa o plugin.
insertSelector 'body' Seletor CSS do elemento onde a seção será inserida.
insertMethod 'beforeend' Posição de inserção: beforeend, afterbegin, beforebegin, afterend.
title 'Perguntas frequentes' Título da seção (coluna esquerda).
titleDot true Exibe o ponto colorido após o título.
description Texto padrão Parágrafo descritivo abaixo do título.
allowMultipleOpen false Se true, permite múltiplos itens abertos ao mesmo tempo.
openFirstItem false Se true, o primeiro item começa aberto.
animationDuration 320 Duração da animação de abertura/fechamento em milissegundos.
items Array de 6 exemplos Array de objetos { question, answer }.
colors Paleta verde/escura Objeto com as cores do plugin (ver abaixo).
