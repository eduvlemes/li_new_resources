Instagram Feed Plugin

Exibe o feed do Instagram com header de perfil, grid responsivo e lightbox — powered by Behold.so.

Demo ao vivo

O feed abaixo é carregado em tempo real via Behold.so. Clique em qualquer post para abrir o lightbox.

API Pública

Use window.InstagramFeed para controlar o plugin via JavaScript:

// Recarrega o feed (útil em SPAs ou após longa inatividade)
window.InstagramFeed.reload();

// Atualiza config e reinicializa
window.InstagramFeed.updateConfig({
    columns: { desktop: 3, tablet: 2, mobile: 1 }
});

// Reinicializa sem alterar a config
window.InstagramFeed.reinit();

// Remove completamente do DOM
window.InstagramFeed.destroy();

Como obter o feedUrl (Behold.so)

1. Acesse behold.so e conecte sua conta do Instagram.
2. Crie um Feed e abra as configurações dele.
3. Copie a Feed URL (ex: https://feeds.behold.so/SEU-ID).
4. Cole na propriedade feedUrl do config abaixo.

Todas as opções disponíveis

Defina window.InstagramFeedConfig antes de carregar o script:

window.InstagramFeedConfig = {

    // ── OBRIGATÓRIO ─────────────────────────────────────────
    feedUrl: 'https://feeds.behold.so/SEU-FEED-ID',

    // ── INSERÇÃO ────────────────────────────────────────────
    insertSelector: 'body',           // seletor CSS do container
    insertMethod:   'beforeend',       // beforeend | afterbegin | etc.

    // ── HEADER DE PERFIL ────────────────────────────────────
    showHeader:       true,
    profile: {
        displayName:     '',              // sobrescreve username do Behold
        showStats:       true,           // publicações / seguidores / seguindo
        postsLabel:      'publicações',
        followersLabel:  'seguidores',
        followingLabel:  'seguindo',
    },
    followButtonText: 'Seguir',
    followButtonUrl:  '',              // ex: 'https://instagram.com/seu_usuario'

    // ── GRID ────────────────────────────────────────────────
    columns: { desktop: 4, tablet: 3, mobile: 2 },
    gap:        '3px',
    imageSize:  'medium',            // 'small' | 'medium' | 'large' | 'full'

    // ── ÍCONES E OVERLAY ────────────────────────────────────
    showVideoIcon:    true,
    showCarouselIcon: true,
    showHoverOverlay: true,

    // ── LIGHTBOX ────────────────────────────────────────────
    showLightbox:      true,
    lightboxImageSize: 'large',
    lightboxLinkText:  'Ver no Instagram',

    // ── TEXTOS ──────────────────────────────────────────────
    loadingText: 'Carregando feed...',
    errorText:   'Não foi possível carregar o feed.',
    emptyText:   'Nenhum post encontrado.',

    // ── CORES (todas opcionais — há padrões definidos) ───────
    colors: {
        sectionBackground: '#ffffff',
        headerText:        '#262626',
        iconColor:         '#ffffff',
        followButton:      '#0095f6',
        followButtonText:  '#ffffff',
        // ... demais tokens em resources/instagram_feed.js
    },
};
