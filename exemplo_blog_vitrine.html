Blog Vitrine

Escaneia o rodapé por um link "Blog", remove ele e as postagens seguintes do menu, e adapta o que mostra de acordo com a página: vitrine na home, compartilhamento + relacionados na postagem, listagem completa na página "Blog".

Como usar

Inclua o script no layout global do tema (roda em toda página, então a limpeza do rodapé e a detecção de contexto funcionam em qualquer lugar). Nenhuma configuração é obrigatória — o comportamento é decidido automaticamente pela URL atual: 

<!-- Incluir no layout global do tema -->
<script src="resources/blog_vitrine.js"></script>
  • Página inicial ("/") — vitrine com as últimas homeLimit postagens, com título
  • Página de uma postagem (uma das que estavam depois de "Blog" no rodapé) — compartilhamento + outras postagens (exceto a atual)
  • Página "Blog" (a do link marcador) — todas as postagens, sem título, em grade de 2 colunas
  • Qualquer outra página — só limpa o rodapé, não mostra nada

Referência completa de opções

Defina window.BlogVitrineConfig antes do script. Só é preciso sobrescrever o que quiser mudar.

window.BlogVitrineConfig = {
    active: true,                    // false desativa o plugin completamente

    blogMarkerText: "Blog",          // texto do link que marca o início do blog, no rodapé
    footerSelector: "footer",        // onde procurar o rodapé (cai pro documento todo se não achar)
    footerPagesHTML: null,             // override manual (string HTML), sobrepõe o scan do rodapé real

    contentSelector: ".conteudo",    // container de conteúdo de cada página de post
    excerptLength: 150,              // caracteres do resumo antes do "..."
    insertSelector: ".conteudo .caixa-sombreada", // onde a vitrine/listagem/compartilhamento é inserido
    insertMethod: "beforeend",       // beforeend | afterbegin | beforebegin | afterend
    storageKey: "blog_vitrine_posts_cache", // chave no sessionStorage

    // Página inicial
    homeLimit: 3,                    // quantas das últimas postagens aparecem na home
    homeSectionTitle: "Blog",

    // Página de uma postagem
    relatedLimit: 3,                 // quantas "outras postagens" aparecem, excluindo a atual
    relatedSectionTitle: "Outras postagens",
    shareText: "Compartilhe:",

    // Página "Blog" (listagem completa)
    listingColumns: { desktop: 2, tablet: 2, mobile: 1 },

    // Colunas da vitrine da home e das "outras postagens"
    columns: { desktop: 3, tablet: 2, mobile: 1 },

    colors: {
        titleColor: "#111111",
        cardBackground: "#ffffff",
        cardBorder: "#e5e5e5",
        excerptColor: "#555555",
        shareText: "#333333"
    }
};

API pública

O plugin fica disponível em window.BlogVitrine após inicialização.

// Reinicializar (usa o cache de sessão, refaz fetch só do que faltar)
window.BlogVitrine.reinit();

// Alterar configuração em runtime
window.BlogVitrine.updateConfig({ homeSectionTitle: "Novidades do blog" });

// Remover a vitrine/listagem/compartilhamento da página
window.BlogVitrine.destroy();
